# Porsche Enterprise Dashboard

Dashboard interativo desenvolvido durante a **Aceleração: AI Reports com Excel, GPT Agents e Claude Code**, promovida pela **DIO** em parceria com o **Santander Open Academy**.

O projeto tem como objetivo transformar dados de vendas da Porsche em informações estratégicas por meio de uma interface moderna, intuitiva e responsiva, inspirada em dashboards de Business Intelligence.

A aplicação permite analisar indicadores de desempenho (KPIs), explorar os dados por meio de filtros interativos e identificar padrões de vendas com base em diferentes perspectivas, como modelo do veículo, ano do modelo, cidade e método de pagamento.

## Funcionalidades

* Dashboard responsivo;
* KPIs de vendas;
* Filtros interativos;
* Análise dos modelos mais vendidos por cidade;
* Identificação do ano-modelo com maior volume de vendas;
* Insights automáticos baseados nos dados;
* Interface com foco em UI/UX utilizando Glassmorphism e uma paleta de tons de rosa e roxo.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## Base de dados

Os dados utilizados foram fornecidos durante a aceleração e correspondem à **Planilha Base de Dados Porsche DIO**, utilizada exclusivamente para fins educacionais.
